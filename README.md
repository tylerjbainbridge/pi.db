# Helpful scripts

## To set up

```bash
npm install && npm run init

```

## To fill the db

```bash
npm run start
```

## To reset db

```bash
npm run reset-db && npm run start
```

## To view data in sql view

```bash
npx prisma studio
```

## Run next.js web server

```bash
npm run dev
```
